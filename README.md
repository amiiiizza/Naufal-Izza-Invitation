# Naufal & Izza Digital Wedding Invitation — Interactive Version

Fitur:
- Button Kirim Ucapan bisa dipencet
- Ucapan tamu tampil langsung di halaman utama
- Ucapan tersimpan di browser menggunakan localStorage
- Button Wedding Gift bisa dipencet
- Nomor rekening BCA bisa disalin
- Tombol musik on/off

Catatan backsound:
Karena lagu "Can't Help Falling in Love" berhak cipta, file lagunya tidak disertakan.
Letakkan file audio legal/licensed milikmu di:
public/music/cant-help-falling-in-love.mp3

Cara menjalankan:
1. Extract ZIP ini
2. Buka foldernya di VS Code
3. Jalankan: npm install
4. Jalankan: npm run dev

Cara deploy ke Vercel:
1. Upload folder ini ke GitHub
2. Buka vercel.com
3. Add New Project
4. Pilih repo ini
5. Klik Deploy
